## 🎯 项目简介

**随礼那点事儿**（原：红包装潢助手）是一个专注于中华传统礼仪文化的微信小程序，致力于帮助用户在现代社会中找到得体的随礼金额，理解传统祝福语的文化内涵，了解全国各地的风土人情。

### ✨ 核心功能

#### 🧮 智能随礼计算器
- **双重维度**：基于关系类型（家庭、朋友、同事、上司）和亲疏程度（一般、较熟、很熟）智能计算
- **个性建议**：为每用户提供贴心的金额建议和贴心提示
- **实际反馈**：支持用户反馈真实金额，持续优化算法准确性
- **智能验证**：完善的输入验证，防止误操作

#### 📝 红包祝福翻译器
- **文白转换**：传统祝福语与现代白话的双向转换，让传统文化更易懂
- **智能匹配**：支持精确搜索和模糊匹配，快速找到合适祝福语
- **分类检索**：8大分类标签（全部、事业、生日、婚礼、开业、马年专属、粤语、网上精选）
- **场景指导**：每条祝福语都详细说明适用场合和场景
- **搜索历史**：自动保存最近5次搜索记录，方便重复使用

#### 📚 全国地域习俗库
- **覆盖全国**：涵盖34个省市自治区（含港澳台地区）的完整文化资料
- **详细分类**：每个地区包含红包金额规范、送礼习俗、特色数字偏好
- **数据智能**：支持按需加载，保证应用性能和数据完整性
- **文化特色**：深入挖掘各地区的地域文化特色和传统习俗

#### 💌 用户反馈系统
- **便捷入口**：悬浮反馈按钮，支持自由拖拽，不遮挡主功能
- **分类反馈**：支持功能异常、优化建议、内容补充等多种反馈类型
- **实时推送**：反馈信息实时推送到团队钉钉群，快速响应
- **隐私保护**：纯前端实现，用户数据存储在本地，保护隐私

## 🚀 快速开始

### 前置要求

- 微信开发者工具（推荐版本 2.01.2510260+）
- 微信小程序基础库 3.14.1+
- Node.js 16+（用于工具脚本开发）
- Git（用于版本管理）

### 本地开发

#### 1. 克隆项目
```bash
git clone <项目地址>
cd redpacket-advisor
```

#### 2. 配置环境
```bash
# 打开微信开发者工具
# 选择项目目录: d:/Projects/redpacket-advisor
# 确认 appid 配置正确
# 设置基础库版本为 3.14.1
```

#### 3. 启动开发
```
# 在微信开发者工具中
1. 点击"编译"按钮
2. 查看控制台输出
3. 检查页面功能
```
