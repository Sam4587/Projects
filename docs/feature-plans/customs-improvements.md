## P0级任务（必须完成）

### 1. 地区详情数据补充
- **功能描述**：为34个地区补充更详细的习俗数据
- **数据结构**：
  ```javascript
  {
    id: String,              // 地区ID
    name: String,           // 地区名称
    giftMoney: {           // 红包习俗
      amountRange: String,   // 金额区间
      customs: Array,       // 习俗要点
      tips: String,          // 贴心提示
    },
    giving: {             // 随份子习俗
      relationships: Object, // 各关系建议金额
      customs: Array,         // 习俗要点
      tips: String
    },
    specialOccasions: {    // 特殊场合
      wedding: Object,      // 婚礼习俗
      birthday: Object,      // 生日习俗
      funeral: Object         // 葬礼习俗
    },
    taboo: Array,          // 禁忌
    story: String           // 文化故事
  }
  ```
- **补充内容**：
  - 地区特色饮食习俗
  - 地区禁忌文化说明
  - 地区典型随礼场景详述
  - 地区文化小故事/趣味知识

## 数据管理规范

### 地区数据完整度
根据数据丰富程度划分等级：
- **A级**：包含完整习俗、禁忌、故事（计划中）
- **B级**：包含基础习俗、金额建议（当前状态）
- **C级**：仅包含地区名称（不适用）

### 数据来源规范
1. **官方渠道**
   - 民政部门公开数据
   - 地方志、县志
   - 文化馆官方资料

2. **学术资料**
   - 民俗学论文、专著
   - 传统文化研究文献

3. **用户反馈**
   - 收集用户补充的习俗信息
   - 匿名或匿名处理
