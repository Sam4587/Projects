## 📝 文档维护说明

### 新增文档时

1. 将文档放入合适的子目录
2. 更新本索引文件中对应的章节
3. 确保文档名称具有描述性

### 归档文档时

1. 将旧文档移动到 `archive/old/` 目录
2. 在索引中添加归档记录
3. 更新相关引用

### 废弃文档时

1. 将文档移动到 `archive/deprecated/` 目录
2. 在索引中标注废弃状态
