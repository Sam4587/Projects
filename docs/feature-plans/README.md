## 数据管理规范

### 数据来源原则
1. **有据可查**
   - 所有推荐数据必须有明确的来源
   - 记录数据出处和参考依据
   - 避免随意杜撰

2. **文化尊重**
   - 尊重各地区的传统习俗差异
   - 准确描述文化含义，避免刻板印象

3. **用户导向**
   - 数据以用户实际需求为准
   - 提供多样化的选择，不强加推荐

### 数据更新流程
1. 数据收集 - 从可靠渠道收集
2. 数据验证 - 多方验证数据准确性
3. 数据入库 - 更新到对应模块
4. 文档同步 - 同步更新相关文档
