## 详细说明

### 翻译页面 - 语音输入
**功能描述**：使用微信语音识别 API 实现语音输入
**技术方案**：
- 氃用 `wx.startRecord()` 开始录音
- 氃用 `wx.stopRecord()` 停止录音
- 调用 `wx.cloud.callFunction()` 语音识别
- 支持 5-10 秒录音时长
- 录音状态可视化

### 翻译页面 - 分享卡片高级模板
**功能描述**：更多样式的精美分享卡片
**模板类型**：
- 传统节日风格（春节、中秋）
- 现代简约风格
- 可爱卡通风格
- 文艺清新风格
- 自定义背景色

### 地域习俗页面 - 地区地图展示
**功能描述**：用地图形式展示34个地区分布
**实现要点**：
- 使用 SVG 地图（简化版）
- 热门地区高亮显示
- 点击地区跳转到详情
- 支持缩放交互

### 地域习俗页面 - 文化小故事
**功能描述**：增加地区文化趣味性
**内容示例**：
- 地区特色美食
- 历史名人故事
- 民间传说
- 趣味习俗

### 商业化功能 - 广告位集成
**功能描述**：接入微信广告变现
**广告类型**：
1. **Banner 广告**：
   - 首页 Banner
   - 翻译页面 Banner
   - 习俗页面 Banner

2. **激励视频**：
   - 深度解读功能解锁
   - 视频观看 30 秒后解锁

3. **原生插屏**：
   - 翻译完成时显示
   - 习俗浏览 3 次后显示

### 商业化功能 - 深度解读付费内容
**功能描述**：付费解锁高级习俗解读
**付费内容示例**：
- 地区专属风水分析
- 深度习俗文化解析
- 专属大师解读
