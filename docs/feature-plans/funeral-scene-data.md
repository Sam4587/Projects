# 葬礼场景文档

## 场景说明

### 场景名称
葬礼（白事）

### 场景特点
- **特殊性**：白事是人生中最庄重的场合之一
- **文化敏感性**：涉及传统丧葬文化、禁忌和礼仪
- **地域差异**：不同地区对葬礼习俗有显著差异
- **金额敏感性**：金额过低被认为不尊重，过高则可能造成压力

## 数据依据说明

### 推荐吉利数字的来源

本产品中葬礼场景推荐吉利数字 **66、88、168、188、200** 的依据：

1. **传统数字寓意**
   - **6 (六)**：代表顺利、平安，在白事中表示逝者一路走好、顺利往生
   - **8 (八)**：代表发财、富贵，寓意祝福家属后生活富足
   - **9 (九)**：代表长久、久长，表达对逝者的怀念之情长存

2. **参考来源**
   - **民俗传统**：中国传统文化中，6、8、9 是常用的吉利数字
   - **地域习俗**：多数地区在白事中也偏好这些数字作为礼物金额
   - **实用考量**：这些数字在百位和千元级别都是合理的、易于接受的

3. **避免的数字**
   - **4**：谐音"死"，在白事中极为忌讳
   - **7**：谐音"气"、"起"，被认为不吉利

4. **地区数据支撑**
   各地区习俗数据（北京、天津、河北等34个地区）中的 `preferredNumbers` 均包含这些数字
   系统根据用户选择的地区动态调整推荐金额基数

## 文化参考

### 葬礼文化要点

1. **白事礼仪**
   - 白色为主色调，代表哀悼
   - 金额应尊重逝者家庭经济状况
   - 避免过于张扬的包装

2. **地域差异**
   - 北方地区：更注重礼仪规范，金额相对稳定
   - 南方地区：更注重实用性和人情往来
   - 沿海地区：可能有更复杂的文化习俗

3. **与一般场合的区别**
   - 葬礼不属于红包"范畴，通常称为"帛金"、"奠仪"
   - 更强调哀思和尊重，而非喜庆氛围
   - 建议使用更庄重的语言和包装

## 建议优化方向

### 1. 场景化处理
- 考虑将"葬礼"独立为场景，与一般场合区分
- 在推荐依据中明确说明是白事场合
- UI 上使用更庄重的色调和设计

### 2. 数据补充
- 可以收集各地区的白事习俗数据
- 记录地区的白事禁忌（如某些地区忌讳的数字或颜色）
- 提供白事礼仪建议（服装颜色、送礼时机等）

### 3. 用户引导
- 在选择葬礼场景时增加提示
- 说明白事与一般场合的文化差异
- 引导用户尊重逝者家庭意愿
