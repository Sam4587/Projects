## 🧪 测试计划

### 测试环境

- 真机测试（iOS + Android）
- 基础库版本：3.14.1

### 测试用例

#### 场景1：随礼计算器完整流程
```
1. 打开小程序 → 默认进入计算器
2. 选择"家人亲戚" → 关系选项高亮
3. 选择"关系密切" → 亲疏程度选项高亮
4. 点击"选择场合" → 弹出场合选择器
5. 选择"生日" → 场合名称更新为"生日"
6. 点击"选择地域" → 弹出地域选择器
7. 选择"北京" → 地域名称更新
8. 点击"预算设置" → 弹出预算设置
9. 输入最低500、最高1000 → 计算结果实时更新
10. 点击"反馈历史" → 显示历史记录列表
```

#### 场景2：红包翻译器完整流程
```
1. 点击 TabBar "红包翻译" → 进入翻译页面
2. 输入"恭喜发财" → 翻译按钮启用
3. 点击翻译 → 显示翻译结果
4. 点击"祝福语库" → 显示祝福语列表
5. 选择"健康"分类 → 筛选健康类祝福语
6. 点击某条祝福语 → 自动填入输入框
7. 输入"福"进行搜索 → 显示相关祝福语
8. 点击复制按钮 → 提示"复制成功"
9. 查看历史记录 → 显示最近翻译
```

#### 场景3：地域习俗完整流程
```
1. 点击 TabBar "地域习俗" → 进入习俗页面
2. 浏览地域列表 → 显示所有地域
3. 点击"华北地区" → 显示该地区习俗详情
4. 选择"婚嫁"分类 → 筛选婚嫁习俗
5. 输入"彩礼"搜索 → 显示相关内容
```

#### 场景4：数据统计查看
```
1. 通过菜单进入"数据统计" → 显示统计数据
2. 查看随礼总额 → 显示正确金额
3. 查看关系分布 → 显示图表
4. 切换时间范围 → 数据正确更新
```
