## 📖 文档编码说明

### 文件编码设置

所有文档均使用 **UTF-8** 编码。

### 如果遇到乱码问题

**方法1: 编辑器设置**
- VSCode: 右下角选择UTF-8
- Notepad++: 编码 → 选择UTF-8

**方法2: 使用命令行查看**
```bash
# Windows
chcp 65001
type filename.md

# Linux/Mac
file filename.md
```

**方法3: 转换编码**
```bash
# 将GBK文件转为UTF-8
iconv -f GBK -t UTF-8 input.md -o output.md
```
